任务1 认识HTML

网站（WebSite）

网站(英文名字为website)是指在因特网上，根据一定的规则，使用HTML等工具制作的用于展示特定内容的相关网页的集合。简单地说，网站是一种通讯工具，就像布告栏一样，人们可以通过网站来发布自己想要公开的资讯，或者利用网站来提供相关的网络服务。人们可以通过网页浏览器来访问网站，获取自己需要的资讯或者享受网络服务。

  • 网站类型
    •
    1. 政府网站
    2. 商业网站
    3. 企业网站
    4. 教育科研机构网站
    5. 个人网站
    6. 非盈利机构网站

  • 网站结构
    •
    1. 线性结构
    2. 二维表结构
    3. 层次结构

网页（Web Page）

网页是构成网站的基本元素，是承载各种网站应用（如：信息浏览、视频点播、资源下载、电子邮件等）的平台。通俗地说，您的网站就是由网页组成的，如果您只有域名和虚拟主机而没有制作任何网页的话，您的客户仍旧无法访问您的网站。 网页是一个包含HTML标签的纯文本文件，它可以存放在世界某个角落的某一台计算机中，是万维网中的一“页”，是超文本标记语言格式（标准通用标记语言的一个应用，文件扩展名为.html或.htm）。网页通常用图像档来提供图画。网页要通过网页浏览器来阅读。其中，在浏览器窗口中首先打开的默认页面称为该网站的首页或主页（Home Page），网站首页一般命名为index和default。

  • 网页文件的类型
    •
    1. 静态网页：.html或.htm或.shtml
    2. 动态网页：.asp或.aspx或.php或.jsp

  • 网页的分类
    •
    1. 静态网页：运行于客户端的程序、网页、组件，属于静态网页，如HTML、Flash、JavaScript、VBScript等
    2. 动态网页：运行于服务器端端的程序、网页、组件，属于动态网页，如ASP、PHP、JSP、ASP.net、CGI等

  • 构成网页的元素
    •
    1. 基本元素
      • 文字
      • 图片
    2. 其它元素
      • 超链接——是网页最重要的特性，链接源可以是文字或图片，通过它可以实现网站间或网站中各页面间的任意跳转。
      • 表格——常用于网页布局和页面排版
      • 音频——.wav, .mp3, .midi等
      • 视频——.avi, .rm, .mpeg等
      • 动画——.swf, .gif等
      • 表单——网页中应用表单向浏览者提供可交互操作界面，以便服务器收集浏览者的各种资料信息。它和脚本程序主要用于动态网页的设计。
      • 脚本程序——脚本程序主要负责运行和处理浏览者向服务器端提交的资料信息（由表单完成的），并向浏览者反馈资料信息处理的结果。它和表单主要用于动态网页的设计。

HTML

HTML（HyperText Markup Language，超文本标记语言），是一种编写超文本文档的简单标记语言。所谓超文本，是因为它可以加入图片、声音、动画、影视等内容，用HTML编写的超文本文档称为HTML文件。每一个HTML文档都是一个静态的网页文件，里面包含许多HTML标记代码，这些代码并不是一种程序语言，它只是一种排版网页中信息显示位置的标记语言，自1990年以来一直被用作WWW的信息表示语言，简单易学，独立于各种操作系统平台，通过浏览器解释执行。

HTTP（HyperText Transfer Protocol，超文本传输协议)是互联网上应用最为广泛的一种网络协议。所有的WWW文件都必须遵守这个标准。HTTP是一个客户端和服务器端请求和应答的标准（TCP）。客户端是终端用户，服务器端是网站。通过使用Web浏览器、网络爬虫或者其它的工具，客户端发起一个到服务器上指定端口（默认端口为80）的HTTP请求。

HTML文档结构基本标记

  • <HTML>……</HTML>标记
    •
    <html>与</html> 标签限定了文档的开始点和结束点，在它们之间是文档的头部和主体。

  • <HEAD>……</HEAD>标记
    •
    <HEAD>……</HEAD>标记只能出现一次，其中可以包含的标记有：
    1. <BASE>标记——可以包含多个，可以不成对出现
    2. <LINK>标记——可以包含多个，可以不成对出现
    3. <META>标记——可以包含多个，可以不成对出现，其中内容有如：
      • name项：常用的选项有keywords(关键字)， description(网站内容描述)， author(作者)， robots(机器人向导)等。
      • http-equiv项：可用于代替name项， 常用的选项有Expires(期限)， Pragma(cache模式)， Refresh(刷新)， Set-Cookie(cookie设定)， Window-target(显示窗口的设定)， content-Type(显示字符集的设定)等。
      • content项：根据name项或http-equiv项的定义来决定此项填写什么样的字符串。
    4. <SCRIPT>标记——可以包含多个，必须成对出现
    5. <STYLE>标记——可以包含多个，可以不成对出现
    6. <TITLE>标记——只能包含一个，必须成对出现

  • <BODY>……</BODY>标记
    •
    <BODY>……</BODY>在HTML文档中只能出现一次，用于标识HTML文档的主体部分。在<BODY>……</BODY>标记之间可包含众多的其他标记和网页内容。其常用属性如下表所示：
    属性功能
    BGCOLOR设置背景颜色
    BACKGROUND设置背景图片，若已经设置了背景颜色，则背景图在上
    TEXT设置文本颜色
    LINK设置超链接颜色
    VLINK设置已使用的超链接颜色
    ALINK设置正在被点击的超链接颜色
    LEFTMARGIN设置页面的左边距
    TOPMARGIN设置页面的顶边距

任务4 建立超链接

建立超链接

格式：<A Href="URL">链接源</A>

TARGET属性的取值及含义
属性值含义
 _blank在新窗口中打开被链接文档
 _parent默认。在相同的框架中打开被链接文档
 _self在父框架集中打开被链接文档
 _top在整个窗口中打开被链接文档跟框架集结构相关
框架名称在指定的框架中打开被链接文档

超链接的应用

  1. 本地链接：链接目标为本地计算机的不同文件的超链接称为本地链接，本地链接通常用相对路径来表示链接目标的位置，所以又称相对URL链接。
  2. 网络链接：链接目标为远程网络服务器上的资源的超链接称为网络链接，网络链接通常用绝对路径来表示链接目标的位置，要有完整的URL地址，包括协议名，主机名，文件所在主机上的路径以及文件名，所以又称绝对URL链接。
  3. 书签链接：
    1. 建立书签

      格式：<A NAME="书签名">……</A>

    2. 建立书签链接

      格式1：页内书签链接
      <A HREF="#书签名">链接源</A>

      格式2：外页书签链接
      <A HREF="外页URL#书签名">链接源</A>

  4. 电子邮件链接：

    格式：<A HREF="mailto:邮件地址?subject=邮件主题>链接源</A>

关于路径

绝对路径：也称绝对URL，它给出目标文件的完整URL地址，包括传输协议在内，如果链接的文件位于外部服务器上，则必须使用绝对路径。
一个完整的URL地址的一般格式为：
Protocol://hostname[:port]/path

相对路径：也称相对URL，相对路径又分为根相对路径和文档相对路径，根相对路径总是以站点根目录“/”为起始目录，写起来比较简单，但大多数情况下，不建议使用这种链接形式。文档相对路径是以当前文档所在路径为起始目录，进行相对的文档查找。一个相对路径不包括协议和主机地址信息，通常只包含文件夹名和文件名，甚至只有文件名。
1. “/” —— 根路径
2. “./” —— 当前目录（可省）
3. “../” —— 上一级目录

 a:visited{}已访问的链接
 a:hover{}当有鼠标悬停在链接上

任务6 建立表单

表单常识

表单是在网页中向访问都提供的一种交互式操作界面，主要作用是向访问者收集各种资料信息，并将这些资料信息提交给网络服务器的CGI（通用网关接口）程序进行处理，使网页具有交互操作的功能。

表单包括两个部分：一部分是HTML源代码用于描述表单（例如，域，标签和用户在页面上看见的按钮），另一部分是脚本或应用程序用于处理提交的信息（如CGI脚本）。

定义表单

格式：<FORM ACTION="URL" METHOD="GET|POST" TARGET="...">. . .</FORM>

表单的输入控件

格式：<INPUT TYPE="输入控件类型">

表单常用输入控件类型
类型含义
text指定表单控件类型为单行文本框
password指定表单控件类型为密码框
file指定表单控件类型为文件域
radio指定表单控件类型为单选框
checkbox指定表单控件类型为复选框
submit指定表单控件类型为提交按钮
reset指定表单控件类型为重置按钮
button指定表单控件类型为普通按钮

表单输入控件的常用属性
属性功能适用于
TYPE指定表单输入框的类型所有控件，必选
NAME指定各输入控件的名称，可为CGI引用所有控件
SIZE设置输入框的长度文本框、密码框和文件域
VALUE输入控件提交的值文本框、密码框、文件域、单选框和复选框
设置按钮上的显示文字提交、重置和普通按钮
CHECKED设置单、复选框初始状态，有此为选中态单选框和复选框
MAXLENGTH设置输入控件中可输入的最大长度文本框、密码框

<FRAMESET>标记

<FRAMESET>...</FRAMESET>标记用于在框架网页中划分浏览器窗口的多个不同显示区域，取代了普通网页中的<BODY>...</BODY>标记。

<FRAMESET>标记的常用属性
属性功能
ROWS用于在浏览器窗口以像素或百分比为单位从上到下划分多个不同的显示区域
COLS用于在浏览器窗口以像素或百分比为单位从左到右划分多个不同的显示区域
FRAMEBORDER用于指定是否显示边框：0——不显示，1——显示
BORDER设置框架边框的粗细
BORDERCOLOR设置框架边框的颜色

<FRAME>标记

<FRAME>标记用于指定在</FRAMESET>标记中被划分的具体的多个不同显示区域及其中所包含的网页。

<FRAME>标记的常用属性
属性功能
SRC用于指定当前显示区域中初始时显示的网页
NAME用于指定当前显示区域的子窗口名称
NORESIZE用于指定多个不同的显示区域是否可以通过拖动框架边框在浏览器中调整显示区域的大小

典型框架结构应用

代码如下：
<HTML>
    <HEAD>
        <TITLE>框架网页应用</TITLE>
    </HEAD>

    <FRAMESET ROWS="15%,*">
        <FRAME SRC="#">
        <FRAMESET COLS="20%,*">
            <FRAME SRC="#">
            <FRAME SRC="#" NORESIZE NAME="content">
        </FRAMESET>
    </FRAMESET>
</HTML>

    属性功能
    SRC指定声音或视频文件的URL

    HEIGHT设置播放音频或视频文件的控制面板的高度。取值为正整数（PX）或百分数

    HIDDEN设置播放音频或视频文件的控制面板是否显示。TURE — 显示；FALSE — 不显示
    CONTROLS设置播放音频或视频文件的控制面板样式。console — 正常面板；smallconsole — 较小面板；playbutton — 只显播放按钮；pausebutton — 只显暂停按钮；stopbutton — 只显停止按钮；volumelever — 只显音量调节按钮

任务9 应用CSS

CSS基本语法

CSS的定义由三个部分构成：选择符、属性和属性的取值。
格式：
选择符{属性:值}

说明：
  1. 选择符可以是多种形式，一般为定义样式的HTML标记，用于定义HTML标记的属性和取值，属性和取值之间用冒号分隔
  2. 属性的取值由多个单词构成时，必须用引号将其属性的取值引起来
  3. 如果需要对一个选择符指定多个属性时，多个属性和值之间用分号隔开，书写时，为了方便阅读，可以采用分行的书写格式
举例：
p{
    text-align:center;
    color:black;
    font-family:"sans serif";
}

常用的几种选择器

1.基础选择器
选择器含义示例
*通用元素选择器，匹配任何元素 * { margin:0; padding:0; }
E标签选择器，匹配所有使用E标签的元素p { font-size:2em; }
.info和E.infoclass选择器，匹配所有class属性中包含info的元素.info { background:#ff0; }

p.info { background:#ff0; }
#info和E#infoid选择器，匹配所有id属性等于footer的元素#info { background:#ff0; }

p#info { background:#ff0; }

2.组合选择器
选择器含义示例
E,F多元素选择器，同时匹配所有E元素或F元素，E和F之间用逗号分隔Div,p { color:#f00; }
E F后代元素选择器，匹配所有属于E元素后代的F元素，E和F之间用空格分隔#nav li { display:inline; }

li a { font-weight:bold; }

3.<A>标记的几个伪类选择器
伪类含义
 a:link{}未访问的链接
 a:visited{}已访问的链接
 a:hover{}当有鼠标悬停在链接上
 a:active{}被选择的链接

  1. 链接到CSS外部样式表
  2. CSS内部样式表
  3. 导入到CSS外部样式表
  4. CSS内嵌样式表

链接到CSS外部样式表

格式：
<LINK REL="stylesheet" TYPE="text/css" HREF="外部样式表文件URL">
属性 取值 功能
HREF 外部样式表文件URL 指定外部样式表的URL，不能省
REL stylesheet 用于设置链接类型
TYPE text/css 用于设置链接内容的类型

举例：
<HTML>
    <HEAD>
        ……
        <LINK REL="stylesheet" TYPE="text/css" HREF="mystyle.css">
        ……
    </HEAD>

    <BODY>
        ……
    </BODY>
</HTML>

CSS内部样式表

格式：
<STYLE TYPE="text/css">内部样式表<STYLE>

举例：
<HTML>
    <HEAD>
        ……
        <STYLE TYPE="text/css">
            hr{color:red;}
            p{margin-left:9px;}
         </STYLE>
        ……
    </HEAD>

    <BODY>
        ……
    </BODY>
</HTML>

格式（注意其中所处位置和后面的分号）：
@import "外部样式表文件URL";

举例：
<HTML>
    <HEAD>
        ……
        <STYLE TYPE="text/css">
            @import "mystyle.css";
            hr{color:red;}
            p{margin-left:9px;}
        </STYLE>
        ……
    </HEAD>

    <BODY>
        ……
    </BODY>
</HTML>

内嵌样式是直接将在HTML文件的标记中通过“STYLE”属性来定义的样式。内嵌样式可以直接对某个标记单独定义样式，使用起来简单、灵活、方便。

举例：
<HTML>
    <HEAD>
        ……
    </HEAD>

    <BODY>
        ……<br>
        <P STYLE="color:black;margin-left:30px;font-weight:bolder;">内嵌样式应用于<P>标记。</p>
        ……<br>
        <DIV STYLE="background-color:RED;text-indent:2em;width:100px;float:left;color:#fff;">内嵌样式应用于<DIV标记>。</DIV>
        ……
    </BODY>
</HTML>
